
# 项目背景
母婴类小程序：打造一款服务于产后妈妈产后恢复相关服务的线上线下相结合平台，齐全的会所详情，产后相关服务和客户评价，每天都有优惠券领取，用户通过快速预约，会所的营养师即可与客户取得联系，并进行相关评价

# 产品定位
专注服务与妈妈产后相关服务的线上预约，订单追踪服务评价的小程序。通过线下营养师的推广，精准推送给客户群体，实现线上线下一体化服务，满足客户的需求

# 客户群体
处于备孕，怀孕和产后的妈妈
